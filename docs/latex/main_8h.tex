\doxysubsection{Detailed Description}
Заголовочный файл с точкой входа и HTTP-\/обработчиком для сервера на базе Mongoose. 

Этот файл содержит объявления\+:
\begin{DoxyItemize}
\item Основной функции {\ttfamily \doxylink{main_8h_ae66f6b31b5ad750f1fe042a706a4e3d4}{main()}}, запускающей HTTP-\/сервер с помощью библиотеки Mongoose.
\item Вспомогательной функции {\ttfamily ev\+\_\+handler()} — обработчика событий HTTP-\/запросов.
\end{DoxyItemize}

Программа инициализирует Mongoose, запускает сервер на порту {\ttfamily 7979} и обрабатывает HTTP-\/запросы\+:
\begin{DoxyItemize}
\item GET-\/запросы ({\ttfamily /api/get})
\item POST-\/запросы ({\ttfamily /api/post})
\item Отдача HTML-\/файлов ({\ttfamily /})
\end{DoxyItemize}

В случае некорректного маршрута клиенту возвращается ошибка 500.
