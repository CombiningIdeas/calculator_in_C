Calculator in C

Проект представляет собой веб-калькулятор, реализованный на языке C с использованием библиотеки <a href="https://github.com/cesanta/mongoose" >Mongoose</a> для запуска веб-сервера. Интерфейс написан на HTML. Серверная часть выполняет вычисления, обрабатывая POST и GET запросы от клиента. <br/>

⚙️ Сборка проекта

Для сборки сервера будет использоваться Makefile. Но тут так же будут указана часть команд, которые можно вручную написать в консоль для изменения каких-либо параметров.
К примеру установить хост и порт можно через консоль:

export HOST=... PORT=...

Проверить их так же можно через консоль:

echo $HOST $PORT

Сборка, пересборка, запуск и очистка проекта с помощью make-команд:

🔨 1. Сборка проекта

Сборка проекта:

make

Эта команда скомпилирует .c-файлы в .o-файлы;

🧹 2. Очистка проекта

Удаления всех объектных файлов, кроме скомпилированного .o-файла:

make clean

Для полной очистки (включая финальный .o-файл):

make all-clean

🔁 3. Перекомпиляция проекта

Для удаления старого и создания нового итогового .o-файл:

make rebuild

🚀 Запуск проекта

Запуск проекта(сервера):

make run

💡 Дополнительно

  • Серверная часть обрабатывает математические выражения, переданные из формы на index.html, и возвращает результат пользователю.
  • Используется HTML-форма для отправки запроса, JavaScript и CSS отсутствуют в основной логике вычислений.
  • Расширенная документация может быть сгенерирована с помощью Doxygen из backend/Doxygen/.
  • Использование автоматичской документации проекта на основе исходного кода через Doxygen:
    doxygen -g Doxyfile
    Подробное описание настроек Doxygen-файла для проекта есть в другом моем github проекте:

    Для отключения документирования папки mongoose в файле Doxygen мы написали в одном флаге следующее значение:
    EXCLUDE = mongoose/
